# LinkedIn Post Generator

A Streamlit app that generates professional LinkedIn posts, reviews them, and improves them automatically using a LangGraph workflow.

## Features

- Generate LinkedIn posts from any topic
- Review the post automatically
- Improve rejected drafts through an iterative loop
- Uses web search when fresh context is useful
- Clean Streamlit UI
- Keeps API keys hidden
